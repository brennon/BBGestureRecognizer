BBUIGestureRecognizerTargetActionWrapper

A BBUIGestureRecognizerTargetActionWrapper wraps both a target instance of any class and a method to be called on that instance. BBUIGestureRecognizerTargetActionWrapper conforms to the BBUIGestureRecognizerTargetAction protocol, and an instance of a BBUIGestureRecognizerTargetActionWrapper is meant to be used as the registeredAction on a BBUIGestureRecognizer.
